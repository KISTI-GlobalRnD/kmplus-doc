## 1. 개요 (Overview)

**KnowledgeMatrix Plus (이하 KM+)**는 논문 및 특허 데이터베이스를 활용하여 연구 동향과 네트워크 분석을 수행할 수 있도록 지원하는 **데이터 전처리 및 행렬 생성 전문 소프트웨어**입니다.

본 소프트웨어는 한국과학기술정보연구원(KISTI)이 개발하였으며, 빅데이터 시대에 전문적인 지식 수준이 없는 일반인도 과학기술 정보를 쉽게 분석하고 활용할 수 있도록 사용자 편의성을 극대화하였습니다.

### 1.1 개발 배경 및 목적

네트워크 분석 과정에서 가장 많은 시간과 노력이 소요되는 단계는 '데이터 정제'와 '분석 도구에 맞는 입력 데이터 생성'입니다. 기존의 해외 과학계량분석 도구(VantagePoint 등)는 한글 처리가 어렵거나 데이터 전처리 기능이 미흡하다는 한계가 있었습니다.

KM+는 이러한 문제를 해결하기 위해 다음과 같은 목적으로 개발되었습니다.

* **한글/영문 데이터 처리:** 영어뿐만 아니라 한글 자연어 처리를 완벽하게 지원하여 국내 연구 환경에 최적화.
* **분석 효율성 증대:** 복잡한 전처리 과정을 자동화하여 연구자가 데이터 해석과 통찰 도출에 집중할 수 있도록 지원.
* **호환성 강화:** 다양한 오픈 소스 네트워크 분석 도구(SNA S/W)와 데이터를 원활하게 연계.

## 2. 주요 기능 및 특징 (Key Features)

KM+는 원시 데이터(Raw Data)를 입력받아 최종 분석 도구에서 활용 가능한 형태로 변환하는 전 과정을 지원합니다.

### 2.1 강력한 데이터 전처리 (Data Pre-processing)

* **형태소 분석:** Apache OpenNLP(영문)와 아리랑 형태소 분석기(한글)를 탑재하여, 논문/특허의 요약문이나 일반 텍스트에서 명사(구)를 정확하게 추출합니다.
* **데이터 정제:** 데이터 축소 및 확장, 불용어 처리, Stemming(어근 추출), 시소러스(유의어) 처리를 통해 데이터를 표준화합니다.
* **중복 제거 및 집합 관리:** 합집합, 교집합 만들기 기능을 통해 프로젝트 간 데이터를 효율적으로 병합하거나 비교할 수 있습니다.

### 2.2 다양한 행렬(Matrix) 생성

* **1-mode Matrix:** 전통적인 Raw Record 기반의 동시 출현 행렬뿐만 아니라, 특정 필드(예: 키워드, 주제 분야) 기준의 행렬 생성을 지원합니다.
* **2-mode Matrix:** '저자-키워드', '국가-주제분류' 등 행과 열이 다른 속성을 가진 2-mode 네트워크 행렬을 생성하여 더욱 풍부한 관계 분석이 가능합니다.
* **3D Matrix:** 3개의 필드를 이용한 입체적인 데이터 구조 분석을 지원합니다.

### 2.3 외부 분석 도구와의 완벽한 연계 (Export)

생성된 데이터는 별도의 변환 과정 없이 대표적인 네트워크 분석 및 시각화 소프트웨어에서 즉시 활용할 수 있습니다.

* **지원 포맷:** Excel, CSV, .NET/CLU/VEC (Pajek), .GEXF (Gephi), VOSviewer 등 [cite: 475-478].
* **속성 데이터 포함:** 네트워크 분석 시 노드의 속성(레코드 수)이나 집단(Partition) 정보까지 함께 출력하여 분석의 깊이를 더합니다.

### 2.4 대용량 데이터 처리 성능

* Web of Science 기준 약 **15,000건**의 논문 데이터를 안정적으로 처리할 수 있으며, 데이터 형식에 따라 최대 수십만 건의 데이터 처리가 가능합니다.
* 2014년 파일럿 버전 대비 Import, Export 및 행렬 계산 속도가 획기적으로 개선되었습니다.

## 3. 분석 프로세스 (Workflow)

KM+를 활용한 정보 분석은 다음과 같은 흐름으로 진행됩니다 [cite: 684-702].

1. **데이터 수집 (Import):** Web of Science, Scopus, NDSL(논문/특허), ScienceDirect, G-Pass 등 다양한 DB의 데이터를 불러옵니다.
2. **데이터 전처리 (Pre-processing):** 형태소 분석, 시소러스 적용, 불용어 처리 등을 통해 분석에 적합한 형태로 데이터를 가공합니다.
3. **통계 분석 (Statistics):** 연도별, 국가별, 기관별 논문/특허 수 등 기초 통계 데이터를 시각화(그래프)하여 확인합니다.
4. **행렬 생성 (Matrix Generation):** 연구 목적에 맞는 1-mode 또는 2-mode 행렬을 생성합니다.
5. **시각화 및 심화 분석 (Export & Analysis):** 생성된 행렬을 Pajek, VOSviewer, Gephi 등으로 내보내어 네트워크 구조를 시각화하고 심층 분석을 수행합니다.

## 4. 시스템 운영 환경 (System Requirements)

KnowledgeMatrix Plus는 개인용 PC에서 독립적으로 작동하는 스탠드얼론(Stand-alone) 소프트웨어로, 별도의 복잡한 설치 과정 없이 압축 해제 후 즉시 실행이 가능합니다.

### 4.1 권장 사양

* **CPU:** Pentium 4 이상
* **운영체제 (OS):** Window / IOS 지원
* **메모리 (RAM):** 3GB 이상 권장
  * *참고: 대용량 데이터 처리를 위해 넉넉한 메모리 확보를 권장합니다.*
* **설치 용량:** 약 180 MB

### 4.2 설치 및 실행 방법

1. 제공된 압축 파일을 다운로드합니다.
2. 원하는 폴더에 압축을 해제합니다.
3. 폴더 내 `KnowledgeMatrixPlus.exe` 실행 파일을 클릭하여 프로그램을 시작합니다.
