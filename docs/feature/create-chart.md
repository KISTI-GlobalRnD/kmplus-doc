필드 데이터를 기반으로 차트를 생성합니다.<br/>
[메뉴바](../program/menu-bar.md#chart) 또는 [툴바 -> Chart](../program/toolbar.md)를 통해 실행할 수 있습니다.

![차트 선택 화면](/img/screenshots/create-chart-start.png)

1. **Graph Type**을 선택합니다.
2. **Sorting Method**를 선택합니다.
3. **Upper Range Value** 값을 지정합니다. (Optional)
4. **View 3D** 여부를 선택합니다. (Optional)
5. **View Examples**(Legend Box) 노출 여부를 선택합니다. (Optional)
6. **다음** 버튼을 클릭합니다.

![필드 선택 화면](/img/screenshots/create-chart-select-field.png)

1. 차트를 그릴 필드를 선택합니다.
2. **완료** 버튼을 클릭합니다.

![차트 생성 결과 화면](/img/screenshots/create-chart-result.png)
