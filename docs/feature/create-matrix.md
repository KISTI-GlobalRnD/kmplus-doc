## 2D Matrix 생성

2개의 필드를 선택하여 행렬 연산을 수행하고 그 결과를 표로 출력합니다.<br/>
[메뉴바](../program/menu-bar.md#analysis) 또는 [메인툴바](../program/toolbar.md)를 통해 실행할 수 있습니다.

![2D 행렬 생성 화면](/img/screenshots/page_17.png)

1. 행렬 타입을 선택합니다.
2. 대상이 되는 필드를 선택합니다.
3. **완료** 버튼을 클릭합니다.

![2D 행렬 생성 결과 화면](/img/screenshots/page_18.png)

* 2D 행렬 생성 결과

## 3D Matrix 생성

3개의 필드를 선택하여 행렬 연산을 수행하고 그 결과를 표로 출력합니다.<br/>
[메뉴바](../program/menu-bar.md#analysis) 또는 [메인툴바](../program/toolbar.md)를 통해 실행할 수 있습니다.

![3D 행렬 생성 화면](/img/screenshots/3d-matrix-select.png)

1. 필드 선택창에서 각각 1개씩 필드를 선택합니다.
2. **완료** 버튼을 클릭합니다.

![3D 행렬 생성 결과 화면](/img/screenshots/3d-matrix-result.png)

* 3D 행렬 생성 결과
