KM+ 는 자주 사용되는 기능에 대한 바로 가기 버튼을 제공합니다.

![툴바](/img/screenshots/toolbar.png)

| 메뉴 항목                                        | 동작 (Action)                                              |
| :----------------------------------------------- | :--------------------------------------------------------- |
| **Create a project**                             | 버튼을 클릭하여 프로젝트 생성을 실행합니다.                |
| **Save as**                                      | 버튼을 클릭하여 현재 프로젝트를 저장합니다.                |
| **New Dataset**                                  | 버튼을 클릭하여 프로젝트 합집합/교집합을 실행합니다.       |
| **Natural Language Processing**                  | 버튼을 클릭하여 한글 명사, 영문 명사/구 추출을 실행합니다. |
| **Chart**                                        | 버튼을 클릭하여 차트를 생성합니다.                         |
| **Statistical Analysis Thumbnail(Publications)** | 버튼을 클릭하여 논문 통계 분석 썸네일을 볼 수 있습니다.    |
| **Statistical Analysis Thumbnail(Patent)**       | 버튼을 클릭하여 특허 통계 분석 썸네일을 볼 수 있습니다.    |
| **Statistical Analysis (Publications)**          | 버튼을 클릭하여 논문 통계 분석 차트를 볼 수 있습니다.      |
| **Statistical Analysis (Patent)**                | 버튼을 클릭하여 특허 통계 분석 차트를 볼 수 있습니다.      |
| **2D Matrix**                                    | 버튼을 클릭하여 2D Matrix를 실행합니다.                    |
| **3D Matrix**                                    | 버튼을 클릭하여 3D Matrix를 실행합니다.                    |
