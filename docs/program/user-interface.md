KnowledgeMatrix Plus (이하 KM+)를 실행하면 메인 윈도우가 열립니다. 이 화면은 사용자가 분석 프로젝트를 생성하고, 데이터를 관리하며, 다양한 분석 결과를 시각적으로 탐색할 수 있도록 구성되어 있습니다.

KM+의 인터페이스는 상단의 메뉴바, 좌측의 프로젝트 탐색기(Project View), 중앙의 작업 영역(Main Editor), 그리고 다양한 툴바와 컨텍스트 메뉴로 이루어져 있어 직관적인 데이터 분석 환경을 제공합니다.

![화면](/img/screenshots/main_ui.png)

KM+의 메인 윈도우는 크게 다섯 가지 주요 영역으로 구분됩니다:

* **메뉴바 (Menu Bar):** 프로그램의 모든 기능에 접근할 수 있는 최상위 메뉴입니다. 프로젝트 관리, 데이터 전처리, 통계 및 네트워크 분석, 환경 설정 등의 기능을 제공합니다.
* **툴바 (Toolbar):** 자주 사용하는 기능(프로젝트 생성, 저장, 뷰 제어 등)을 아이콘 형태로 제공하여 빠르게 실행할 수 있습니다.
* **프로젝트 뷰 (Project View):** 현재 열려 있는 프로젝트의 구조를 트리(Tree) 형태로 보여줍니다. 필드(Field) 목록, 생성된 그래프, 행렬 등을 계층적으로 관리할 수 있습니다.
* **작업 영역 (Main Workspace):** 선택한 필드의 상세 데이터, 생성된 그래프, 분석 결과 행렬 등이 표시되는 공간입니다. 탭(Tab) 형식으로 여러 작업을 동시에 열어둘 수 있습니다.
* **상태 표시줄 (Status Bar):** 프로그램 하단에 위치하며, 현재 작업 상태나 메모리 사용량 등의 시스템 정보를 표시합니다.

---

## 1. 메뉴바 (Menu Bar)

메뉴바는 KM+의 핵심 기능을 카테고리별로 정리해 두었습니다.

![메뉴바](/img/screenshots/menubar.png)

* **홈 (Home):** 프로젝트 생성(`프로젝트 만들기`), 프로젝트 열기 및 저장, 프로그램 종료 등 파일 및 프로젝트 관리 기능을 수행합니다.
* **전처리 (Pre-processing):** 데이터 분석 전 과정을 지원합니다. 데이터 합집합/교집합 생성, 영문/한글 형태소 분석, 필드 병합 등의 기능을 제공합니다.
* **통계 분석 (Statistics):** 논문 및 특허 데이터에 특화된 기초 통계 분석(연도별, 국가별, 저자별 논문 수 등)과 시각화 기능을 제공합니다.
* **네트워크 분석 (Network Analysis):** 2D/3D 행렬 생성, 네트워크 그래프 시각화, 외부 분석 도구(Pajek, VOSviewer, Gephi)와의 연동 기능을 포함합니다.
* **설정 (Settings):** 데이터 임포트 설정(Import Editor), 시소러스 관리(Thesaurus Editor), 화면 레이아웃 초기화 등을 수행합니다.
* **도움말 (Help):** 매뉴얼, 튜토리얼, 프로그램 정보 등을 확인할 수 있습니다.

## 2. 프로젝트 뷰 (Project View)

좌측에 위치한 프로젝트 뷰는 분석 중인 데이터를 체계적으로 관리하는 탐색기입니다.

![프로젝트 뷰](/img/screenshots/project_view.png)

* **프로젝트 트리:** 생성된 프로젝트와 그 하위의 필드 목록이 트리 구조로 표시됩니다.
* **컨텍스트 메뉴 (우클릭):** 프로젝트나 필드 항목을 마우스 오른쪽 버튼으로 클릭하면 프로젝트 이름 변경, 닫기, 요약 정보 보기 등의 추가 메뉴를 사용할 수 있습니다.
* **데이터 요약 확인:** 각 필드 옆에는 해당 필드에 포함된 레코드 수와 같은 요약 정보가 표시될 수 있습니다.

## 3. 작업 영역 (Main Workspace)

화면 중앙의 작업 영역은 실제 데이터 분석과 시각화가 이루어지는 공간입니다. 사용자가 프로젝트 뷰에서 선택한 항목에 따라 다양한 뷰(View)가 탭 형태로 열립니다.

![작업 영역](/img/screenshots/main_workspace.png)

* **필드 리스트 뷰 (Field List View):** 선택한 필드의 데이터를 리스트 형태로 보여주며, 빈도수 확인, 그룹 생성, 필터링 등의 작업을 수행할 수 있습니다.
* **통계/그래프 뷰 (Chart View):** 분석된 통계 데이터를 막대, 파이, 선 그래프 등 다양한 차트로 시각화하여 보여줍니다.
* **행렬 뷰 (Matrix View):** 생성된 2d 또는 3d 행렬 데이터를 스프레드시트 형태로 표시하며, 데이터 정렬 및 내보내기(Export)가 가능합니다.
* **레코드 뷰 (Record View):** 개별 데이터의 원문 정보(초록, 저자 상세 등)를 확인할 수 있는 상세 창입니다.

## 4. 툴바 및 컨텍스트 메뉴 (Toolbar & Context Menus)

작업 효율성을 높이기 위해 메인 툴바 외에도 각 뷰(View) 내부에는 상황에 맞는 기능 버튼들이 배치되어 있습니다.

![툴바](/img/screenshots/toolbar.png)

* **메인 툴바:** 새 프로젝트, 저장, 글꼴 설정, 창 배치 등 전역적인 기능을 수행합니다.

![퀵메뉴](/img/screenshots/quickmenu.png)

* **뷰 전용 툴바:** 예를 들어, '필드 리스트 뷰' 상단에는 그래프 생성, 그룹 관리, 부분 집합 생성, Stemming(어근 처리) 등 해당 데이터 처리에 특화된 버튼들이 제공됩니다.
