### 툴바

작업 효율성을 높이기 위해 상황에 맞는 기능 버튼이 상단에 배치되어 있습니다.

![필드리스트-툴바](/img/screenshots/fieldlistview-toolbar.png)

| 메뉴항목                         | 동작                                                                                                                                  |
| :------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------ |
| **Create Graph**                 | 그래프 유형을 선택하여 해당 필드의 차트를 생성합니다.                                                                                 |
| **Add Selection to New Group**   | 신규 그룹 만들기 혹은 필드리스트에서 선택한 데이터로 신규 그룹을 생성합니다.                                                          |
| **Add to group**                 | 필드리스트에서 선택한 데이터를 사전에 만든 그룹리스트에서 선택하여 그룹에 포함시킵니다.                                               |
| **Manage groups**                | 신규 그룹 추가(Add), 그룹을 선택하여 기존 그룹 삭제(Delete), 그룹을 선택하여 그룹 이름을 변경(Rename) 합니다.                         |
| **Create Sub-dataset**           | 1) 필드리스트에서 선택한 데이터 혹은 2) 기존의 그룹을 선택하여, 해당 데이터가 포함된 raw record를 추출, 새로운 프로젝트를 생성합니다. |
| **Stemming**                     | 어간을 추출하여 동의어를 자동으로 정렬(stemming), 두 개의 창을 활용하여 편집이 가능합니다.                                            |
| **Apply the Existing Thesaurus** | 사전에 정의한 시소러스 파일을 불러 읽어 데이터를 정제합니다.                                                                          |

### 컨텍스트 메뉴

필드리스트를 마우스 오른쪽 버튼으로 클릭하면 컨텍스트 메뉴가 나타납니다.

![필드리스트-컨텍스트 메뉴](/img/screenshots/fieldlistview-contextmenu.png)

| 메뉴항목                         | 동작                                                                                                                                  |
| :------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------ |
| **Copy**                         | 필드리스트에서 선택한 데이터를 복사합니다. (텍스트 편집기 혹은 엑셀에서 붙여넣기)                                                     |
| **Save as Excel**                | 필드리스트 전체를 엑셀로 저장합니다.                                                                                                  |
| **Add Selection to New Group**   | 신규 그룹 만들기 혹은 필드리스트에서 선택한 데이터로 신규 그룹을 생성합니다.                                                          |
| **Add to group**                 | 필드리스트에서 선택한 데이터를 사전에 만든 그룹리스트에서 선택하여 그룹에 포함시킵니다.                                               |
| **Create Sub-dataset**           | 1) 필드리스트에서 선택한 데이터 혹은 2) 기존의 그룹을 선택하여, 해당 데이터가 포함된 raw record를 추출, 새로운 프로젝트를 생성합니다. |
| **Stemming**                     | 어간을 추출하여 동의어를 자동으로 정렬(stemming), 두 개의 창을 활용하여 편집이 가능합니다.                                            |
| **Apply the Existing Thesaurus** | 사전에 정의한 시소러스 파일을 불러 읽어 데이터를 정제합니다.                                                                          |
