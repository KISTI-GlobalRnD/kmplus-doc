개별 데이터의 원문 정보를 확인할 수 있는 상세 창입니다.

![레코드 뷰](/img/screenshots/recordview.png)

### 컨텍스트 메뉴

레코드뷰를 마우스 오른쪽 버튼으로 클릭하면 컨텍스트 메뉴가 나타납니다.

![레코드 뷰-컨텍스트 메뉴](/img/screenshots/recordview-contextmenu.png)

| 메뉴항목        | 동작                                                                                               |
| :-------------- | :------------------------------------------------------------------------------------------------- |
| **Copy**        | 레코드뷰에서 선택한 셀의 헤더를 포함한 데이터를 복사합니다. (텍스트 편집기 혹은 엑셀에서 붙여넣기) |
| **Save as tsv** | 레코드뷰 전체를 탭 구분 텍스트로 저장합니다. (*.tsv 파일로 저장)                                   |
