행렬 데이터를 스프레드 시트 형식으로 나타내며, 데이터 export 기능을 제공합니다.

![행렬 뷰](/img/screenshots/matrixview.png)

### 툴바

작업 효율성을 높이기 위해 상황에 맞는 기능 버튼이 상단에 배치되어 있습니다.

![레코드 뷰-컨텍스트 메뉴](/img/screenshots/matrixview-toolbar.png)

| 메뉴 항목                        | 동작                                                                                                           |
| :------------------------------- | :------------------------------------------------------------------------------------------------------------- |
| **Filter Number**                | 화면에 표시할 최소 값을 지정합니다. (예: 3으로 설정시 3 미만 값은 보이지 않습니다.)                            |
| **EXPORT**                       | 행렬의 원본 데이터를 내려받습니다. (txt, csv, xlsx 포맷과 pajek, gephi 에서 사용할 수 있는 포맷을 지원합니다.) |
| **Matrix Chart**                 | 행렬 데이터를 기반으로 **차트 뷰**를 실행합니다.                                                               |
| **행 정렬 (Row Field)**          | 행으로 지정된 필드 값을 기반으로 정렬합니다.                                                                   |
| **행 정렬 (Row RecordCount)**    | 행으로 지정된 필드의 레코드 수를 기반으로 정렬합니다.                                                          |
| **열 정렬 (Column Field)**       | 열로 지정된 필드 값을 기반으로 정렬합니다.                                                                     |
| **열 정렬 (Column RecordCount)** | 열로 지정된 필드의 레코드 수를 기반으로 정렬합니다.                                                            |
